# Avatar-Trading-Algorithm
The Avatar Trading Algorithm is a ThinkorSwim Strategy strictly for the S&P Futures market (ES or MES) that utilizes several components:
- Trend
  - ATR Trailing Stop
  - 20, 30, 50, 100 Simple Moving Averages
  - Darvas Box
- Momentum
  - Bollinger Bands
  - MacD
  - TTM Squeeze
- Technical Patterns
